# Riyadh Metro Legal Information

Welcome to the official legal information page for the Riyadh Metro app.

## About the Riyadh Metro App

The Riyadh Metro app is your essential companion for navigating the Riyadh public transportation system. Our app provides a seamless user experience with the following features:

### 🗺️ Metro Map Viewer
View the complete Riyadh Metro network with all lines, stations, and interchanges in an interactive map.
![Metro Map](./images/metro-map.jpg)

### 🔍 Station Search
Easily find any station on the network by name or nearby location.
![Station Search](./images/station-search.jpg)

### 🛣️ Route Planning & Navigation
Plan your journey from any station to destination, with transfer information and estimated travel times.
![Route Planning](./images/route-planning.jpg)

### ⏱️ Real-time Information
Get up-to-date information about operating hours for all stations.
![Station Details](./images/station-details.jpg)
